	\section{Аннотация}
	\sectionmark{Аннотация}
В работе рассматривается задача гарантированного вычисления всех стационарных состояний модели ВИЧ-инфекции Бочарова-Перцева при фиксированных значениях параметров и анализа их устойчивости. Показано, что исходная система семи нелинейных уравнений, неотрицательными решениями которой являются стационарные состояния, сводится к системе двух уравнений. Аналитически локализована область возможных неотрицательных решений. Предложена эффективная технология вычисления всех неотрицательных решений  и анализа их устойчивости. 
Полученные результаты дают вычислительную основу для исследования хронических состояний с помощью модели Бочарова-Перцева.

	\section{Математическая модель противовирусного иммунного ответа}
\label{S2}
Математическая модель ВИЧ-инфекции Бочарова-Перцева сформулирована в \cite{Pertsev_2019} в виде системы нелинейных интегро-дифференциальных уравнений с запаздывающим аргументом. Система описывает скорость изменения во времени концентрации следующих популяций: 

\begin{itemize} 
	\item неинфецированных клеток-мишеней $A(t)$ --- дендритные клетки, макрофаги, CD4+ T-лимфоциты;
	\item латентно инфецированных клеток $L(t)$ --- клетки, незаметно инфецированные для иммунной системы;
	\item инфецированных клеток, производящих вирусные частицы $I(t)$;
	\item зрелых вирусных частиц $V(t)$ --- вирусные частицы, готовые заражать;
	\item наивных неактивированных CD8+ T-лимфоцитов $E_0(t)$;
	\item клеток памяти CD8+ T-лимфоцитов $Q(t)$;
	\item эффекторных цитотоксичных CD8+ T-лимфоцитов $E(t)$.
\end{itemize}
Опишем популяции клеток, играющие важную роль в процессе иммунного ответа, но концентрации которых не будут использоваться в расчетах:
\begin{itemize}
	\item продуктивно-инфецированные клетки до выхода из них новых вирусных частиц $C(t)$---заметны для иммунной системы, но не производят новые вирусные частицы;
	\item незрелые вирусные частицы $U(t)$ --- только что вышедшие из инфицированной клетки новые вирусные частицы, не готовые заражать;
	\item активированные пролиферирующие CD8+ T-лимфоциты $E_1(t)$;	
	\item активированные клетки памяти CD8+ T-лимфоцитов $E_2(t)$.
\end{itemize}
Будем предполагать далее, что все переменные имеют размерность клеток/мл или частиц/мл. \\

Введем следующие вспомогательные переменные:\newline
скорость появления клеток $C$ $$\rho_C(t) = (1-p_L)\gamma_{A,V}A(t)V(t)+\gamma_{A,I}A(t)I(t)+\gamma_{L,I}L(t)I(t)+\alpha_LL(t), $$
скорость секреции клеток $U$ $$\rho_U(t)=\nu_II(t),$$
скорость активации клеток $E_1$ $$\rho_{E_1}(t)=\gamma_{E_0,A,V}E_0(t)A(t)V(t),$$
скорость активации клеток $E_2$
$$\rho_{E_2}(t) = \gamma_{Q,A,V}Q(t)A(t)V(t).$$

Система состоит из двух блоков: блока, описывающего процесс формирования инфецированных и латентно 	инфецированных клеток
{\allowdisplaybreaks
	\begin{eqnarray}
	\frac { d }{ dt } A(t) &=&r_A - \mu_A A(t)-\gamma_{A,V} A(t)V(t)-\gamma_{A,I}A(t)I(t),\label{Model1_1} \\
	\nonumber \\
	%
	\frac { d }{ dt } L(t) &=& -(\mu_L +\alpha_L)L(t)-\gamma_{L,I}L(t)I(t)+p_L\gamma_{A,V}A(t)V(t),\label{Model1_2} \\
	\nonumber \\
	%
	\frac { d }{ dt } I(t) &=& -(\mu_I+\sigma_I\nu_I)I(t)-\gamma_{E,I}E(t)I(t)+e^{-\mu_C\omega_C}\rho_C(t-\omega_C),
	\label{Model1_3} \\
	\nonumber \\ 
	%
	\frac { d }{ dt } V(t) &=&-\mu_VV(t)-\gamma_{A,V}A(t)V(t)-\gamma_{L,V}L(t)V(t)- \label{Model1_4} \\
	&&-\gamma_{C,V}\int_{t-\omega_C}^{t}e^{-\mu_C(t-s)}\rho_C(s)dsV(t)
	+e^{-\mu_U\omega_U}\rho_U(t-\omega_U), 
	\nonumber 
	\end{eqnarray}
	и блока, описывающего иммунный ответ	
	\begin{eqnarray}
	\frac { d }{ dt } E_0(t) &=& 
	r_{E_0}-\mu_{E_0}E_0(t)-\rho_{E1}(t),
	\label{Model1_5} \\
	\nonumber \\
	%
	\frac { d }{ dt } Q(t) &=& -\mu_QQ(t)-\rho_{E_2}(t)+n_{Q_1}\rho_{E_1}(t-\omega_{E_1})+n_{Q_2}\rho_{E_2}(t-\omega_{E_2}),
	\label{Model1_6} \\
	\nonumber \\
	%
	\frac { d }{ dt } E(t) &=& -\mu_EE(t)-p_E\gamma_{E,I}E(t)I(t)+n_{E_1}\rho_{E_1}(t-\omega_{E_1})+
	\label{Model1_7} \\
	&&+n_{E_2}\rho_{E_2}(t-\omega_{E_2}).
	\nonumber 
	\end{eqnarray}
	
	Биологический смысл параметров системы пояснен в Табл. \ref{Table1}. 
	Для биологической  адекватности модели (\ref{Model1_1})---(\ref{Model1_7}) значения  всех этих параметров должны быть неотрицательными.  Более того, интерес представляет только случай, когда все параметры положительные и выполнены
	следующие неравенства:
	\begin{eqnarray}	
	0<\sigma_I<1, \;
	0\leq p_E\leq 1.  \label{HYP1}
	\end{eqnarray}
	
	Далее мы будем предполагать, что  неравенство (\ref{HYP1}) выполнено.

	Отметим, что задача Коши для системы (\ref{Model1_1})---(\ref{Model1_7}) с неотрицательными непрерывными начальными значениями и неотрицательными параметрами глобально разрешима на любом конечном временном интервале \cite{Pertsev_2018}. Для  определения решения при $t>0$ достаточно задать значения 
	\begin{eqnarray}
	A(t)=A^0(t), \, L(t)=L^0(t), \, I(t)=I^0(t), \, V(t)=V^0(t), \nonumber \\
	E_0(t)=E_0^0(t),\, Q(t)=Q^0(t),\, E(t)=E^0(t), \qquad -\omega_{max}\leq t\leq 0, \label{initial}
	\end{eqnarray}
	где $\omega_{max}=\max\{\omega_C,\omega_U,\omega_{E_1},\omega_{E_2}\}$. Все функции в (\ref{initial}) неотрицательные и непрерывные.
	
	
	
	Обозначив вектор переменных системы  (\ref{Model1_1})---(\ref{Model1_7}) через
	\begin{equation}
	U= (A, L, I, V, E_0, Q,E)^T, \label{varU}
	\end{equation}
	будем записывать эту систему в следующем компактном виде:
	\begin{multline}
	\frac { dU}{ dt }(t) =F(U(t),U(t-\omega_C), U(t-\omega_U),U(t-\omega_{E_1}), U(t-\omega_{E_2}),\\\{U(s):t-\omega_C\leq s \leq t\}), 
	\label{Model2}
	\end{multline}
	где	$\omega_C,\omega_U,\omega_{E_1},\omega_{E_2}$ означают соответствующие задержки.

	\section{Задача Коши}
	\subsection{Описание алгоритма решения задачи Коши} \label{sec_3.1}
	Для численного решения задачи Коши для системы (\ref{Model1_1})---(\ref{Model1_7}) в данной работе применим неявную схему второго порядка BDF2 \cite{HNW} на равномерной сетке
	
	$$
	\{t_k=\delta k:\; k=-m_{max}+1,-m_{max}+2,\ldots\},
	$$
	построенной с шагом $\delta>0$ в полуинтервале $[-\omega_{max},\infty)$. Величины $m_\alpha=[\omega_\alpha/\delta]$, $ \alpha\in\{C,U,E_1,E_2\}$, где  $[.]$ означает целую часть числа, являются дискретными аналогами задержек, $m_{max}=max\{m_C,m_U,m_{E_1},m_{E_2}\}$. Пусть $U_k$ --- сеточная функция, аппроксимирующая   $U(t_k)$. Заменим интеграл в уравнении (\ref{Model1_4}) на сумму по формуле трапеции
	$$\int_{t-\omega_C}^{t}e^{-\mu_C(t-s)}\rho_C(s)ds=\sum_{s=k-m_C+1}^{k}\frac{e^{-\mu_C \delta (k-(s-1))} \rho_{C{s-1}}+e^{-\mu_C \delta(k-s)} \rho_{Cs}}{2}\delta.$$
	Тогда система (\ref{Model2}) примет следующий вид:
	\begin{multline}\label{LinBDF2a}
	\frac { 1.5U_k- 2U_{k-1} +0.5U_{k-2}}{\delta} =\\= \widehat{F}(U_k,U_{k-m_C}, U_{k-m_U},\ldots, U_{k-m_{E_2}},\{U_k, U_{k-1}, \ldots U_{k-m_C}\}),\\
	k=1,2,\ldots.
	\end{multline}
	В качестве начальных данных для решения задачи Коши нужно задать значения  $U_{-m_{max}+1}, \ldots, U_{0}$.
	
	Запишем уравнение (\ref{LinBDF2a}) в виде
	\begin{eqnarray}
	G(U_k)=0 \label{G=0},\qquad k = 1,2,\ldots,
	\end{eqnarray}
	где
	\begin{multline}
	G(U_k) = \frac{1.5U_k- 2U_{k-1} +0.5U_{k-2}}{\delta} - \\ - \widehat{F}(U_k,U_{k-m_C}, U_{k-m_U},\ldots, U_{k-m_{E_2}},\{U_k, U_{k-1}, \ldots U_{k-m_C}\}). \nonumber 
	\end{multline}
	Якобиан функции $G(U_k)$ имеет вид 
	\begin{equation}\label{Jacob}
	J_k = \frac{1.5I}{\delta}- D_k, \nonumber
	\end{equation}
	где
	$$
	D_k = \frac { \partial \widehat{F}(U_k,U_{k-m_C}, U_{k-m_U},\ldots, U_{k-m_{E_2}},\{U_k, U_{k-1}, \ldots U_{k-m_C}\})} {\partial U_k }, 
	$$
	а $I$ означает единичную матрицу седьмого порядка. Будем вычислять $D_k$ по формуле
	$$
	D_k=\frac {\widehat{F}(U_k+\epsilon,U_{k-m_C},\ldots )-\widehat{F}(U_k-\epsilon,U_{k-m_C},\ldots )}{2\epsilon }.
	$$
	
	
	На каждом шаге решаем уравнение (\ref{G=0}) относительно $U_k$ методом Ньютона с начальным значением $U_{k-1}$
	$$
	U_{l+1}=U_l-J_l^{-1}\cdot G(U_l)
	$$
	до достижения точности  $||G(U_l)||_2\leq \epsilon_N$.

	\section{Анализ устойчивости}
	\label{S4}
	
	
	Для исследования устойчивости заданного стационарного состояния $\overline{U}$ системы  (\ref{Model2}) при фиксированных значениях параметров представим решение вблизи этого стационарного состояния в виде $U(t) = \overline{U} + \varepsilon U'(t) + o(\varepsilon^2)$, где $\varepsilon$ --- малый параметр. Подставив это представление в (\ref{Model2}) и пренебрегая членами порядка $\varepsilon^2$, для функции $U'(t)$ получим систему следующего вида:
	\begin{equation} 
	\frac { d}{ dt }U'(t) = L_0U'(t)+\int_{t-\omega_C}^t e^{-\mu_C(t-s)}L_1U'(s)ds+\sum_{j\in\{C,U,E_1,E_2\}} L_jU'(t-\omega_j) ,\label{system}
	\end{equation}
	где $L_0$, $L_1$ $L_j$ означают постоянные, квадратные матрицы седьмого порядка.
	%, являющиеся соответственно значениями в стационарном состоянии частных производных
	%$\partial F/\partial U$, \textcolor{red}{$ \frac{1}{e^{-\mu_C(t-s)}}\partial F/\partial U(s)$} и  
	%$\partial F/\partial U(t-\omega_j)$
	% правой части системы (\ref{Model2}).
	
	
	Стационарное состояние $\overline{U}$ асимптотически устойчиво, если любое решение системы (\ref{system}) вида
	\begin{gather}
	U'(t) = \widetilde{U} e^{\lambda t} \label{solution},
	\end{gather}
	где $\widetilde{U}$  --- постоянный десяти компонентный вектор, а $\lambda$ --- число, монотонно убывает при $t \rightarrow \infty$. Подставим (\ref{solution}) в (\ref{system}), получив следующую нелинейную проблему собственных значений:
	\begin{gather}
	A(\lambda)\widetilde{U} = 0 \label{problem},
	\end{gather}
	где 
	
	\begin{gather*}
	A(\lambda) =\lambda I- L_0-\sum_j L_je^{-\lambda\omega_j}-
	\begin{cases}
	\displaystyle
	\frac{1-e^{-(\lambda+\mu_C)\omega_C}}{\mu_C+\lambda}L_1, & |\lambda +\mu_C| \geq 1,\\
	\displaystyle
	\sum_{k=0}^{\infty} \frac{(-1)^k(\lambda+\mu_C)^k\omega_C^{k+1}}{(k+1)!}L_1, & |\lambda +\mu_C|< 1,
	\end{cases}
	\end{gather*}
	где $j\in\{U,C,E_2,E_1\}$, а $I$ --- единичная матрица седьмого порядка. 
	
	Таким образом, исследование асимптотической устойчивости стационарного состояния $\overline{U}$ сводится к решению нелинейной проблемы собственных значений (\ref{problem}) и проверке, что все найденные собственные значения $\lambda$ лежат строго в левой полуплоскости.
	%%%%%%%TODO изменить этот абзац 
	Алгоритмы, которые позволяют решать полную нелинейную проблему собственных значений вида (\ref{problem}), не известны. Более того, вообще говоря, эта проблема имеет бесконечное число решений. Так что мы будем аппроксимировать эту проблему рациональной проблемой собственных значений. Решения рациональной проблемы будут приближениями к решениям исходной проблемы (\ref{problem}), которые можно затем уточнить при помощи какого-либо из локальных методов ньютоновского типа.
	
	
	
	Для поиска $\lambda$ так же как и в пункте \ref{sec_3.1} применим неявную схему второго порядка BDF2 \cite{HNW} на равномерной сетке
	$$
	\{t_k=\delta k:\; k=-m_{max}+1,-m_{max}+2,\ldots\},
	$$
	построенной с шагом $\delta>0$ в полуинтервале $(-\omega_{max},\infty)$. Величины $m_\alpha=[\omega_\alpha/\delta]$, $ \alpha\in\{C,U,E_1,E_2\}$, где  $[.]$ означает целую часть числа, являются дискретными аналогами задержек, $m_{max}=max\{m_C,m_U,m_{E_1},m_{E_2}\}$. Тогда система (\ref{system}) примет следующий вид:
	\begin{multline}\label{LinBDF2_stab_lin}
	\frac { 1.5U'_k- 2U'_{k-1} +0.5U'_{k-2}}{\delta} =\\= L_0 U'_k +\sum_{s=k-m_C+1}^k \frac{\delta}{2}L_1(e^{-\mu_C\delta(k-(s-1))}U'_{s-1}+e^{-\mu_C\delta(k-s)}U'_s) + \\+\sum_{j\in\{C,U,E_1,E_2\}} L_j U'_{k-m_j},
	k=1,2,\ldots.
	\end{multline}
	где $U'_k$ --- сеточная функция, аппроксимирующая   $U'(t_k)$. 
	
	
	Для исследования устойчивости будем искать $U'_k$ в виде $U'_k=\widetilde{U}\mu^k$. Соответственно, $\lambda = \ln \mu / \delta$. Тогда система (\ref{LinBDF2_stab_lin}) примет вид
	\begin{equation} \label{mu}
	\widehat{A}(\mu)\widetilde{U}=0,
	\end{equation}
	где
	\begin{multline} 
	\widehat{A}(\mu)=\frac{1.5 - 2 \mu^{-1} + 0.5 \mu^{-2} }{\delta}  I
	-L_0-\\-\sum_{s=1}^{m_C}\frac{\delta}{2}L_1( e^{-\mu_C\delta (s-1)}\mu^{-(s-1)}+e^{-\mu_C\delta s}\mu^{-s}) -\sum_{j\in\{C,U,E_1,E_2\}} L_j\mu^{-m_j}.\nonumber
	\end{multline}
	
	
	Для всех наших наборов  параметров верно: $m_U\leq m_C\leq m_{E_2}\leq m_{E_1} $.
	Умножим уравнение (\ref{mu}) на $\mu^{m_{E_1}}$. Полученное уравнение можно привести к следующей полиномиальной проблеме: 
	\begin{multline} 
	\label{m_4_equation}
	\mu^{m_{E_1}}\widetilde{U} = \\=\mu^{m_{E_1}-1}C_1\widetilde{U} + \mu^{m_{E_1}-2} C_2 \widetilde{U} +\sum_{s=1}^{m_C} \mu^{m_{E_1}-s}C'_s\widetilde{U} +\sum_{j\in\{C,U,E_1,E_2\} } \mu^{m_{E_1}-m_j}C_{m_j}\widetilde{U}, 
	\end{multline}
	где
	\begin{equation}
	\begin{gathered} 
	C_1 = 2(1.5I - \delta L_0-\frac{\delta^2}{2} L_1)^{-1}, \\ 
	C_2 = -0.5(1.5I - \delta L_0-\frac{\delta^2}{2} L_1)^{-1},   \\ 
	C'_s = (1.5I - \delta L_0-\frac{\delta^2}{2} L_1)^{-1}\delta^2 e^{-\mu_C\delta s}L_1, \quad s=1\ldots m_C-1,\\
	C'_{m_C} = (1.5I - \delta L_0-\frac{\delta^2}{2} L_1)^{-1}\frac{\delta^2}{2} e^{-\mu_C\delta m_C}L_1,\\
	C_{m_j} = (1.5I - \delta L_0-\frac{\delta^2}{2} L_1)^{-1}\delta L_{j}, \quad j\in\{C,U,E_1,E_2\}. \nonumber
	\end{gathered}
	\end{equation}
	Дополним уравнение (\ref{m_4_equation}) тождествами  $\widetilde{U}\mu^j =\widetilde{U}\mu^j, \; j = 1,\ldots,m_{E_1}-1$, таким образом получив систему из $m_{E_1}$ уравнений, которую можно записать в виде: 
	\begin{gather} 
	\mu\widetilde{X} = M\widetilde{X}, 
	\end{gather}
	где 
	\begin{equation*}  
	\widetilde{X} = \left( 
	\begin{array}{c}
	\widetilde{X}_1 \\
	\vdots  \\
	\widetilde{X}_{m_{E_1}}
	\end{array}
	\right), \;
	M = \left(
	\begin{array}{ccc}
	M_{11} & \ldots & M_{1m_{E_1}}\\
	\vdots & \ddots & \vdots \\
	M_{m_{E_1}1} & \ldots & M_{m_{E_1} m_{E_1}}
	\end{array}
	\right). 
	\end{equation*}
	
	Здесь $\widetilde{X}_i = \widetilde{U}\mu^{m_{E_1} - i} $ при $i = 1,\ldots,m_{E_1}$, а матрица $M$ является блочной, блочного порядка $m_{E_1}$ с блоками порядка $7$. Все блоки этой матрицы нулевые, кроме поддиагональных блоков $M_{i+1,i} = I \; (i = 1,\ldots,m_{E_1}-1)$ и некоторых блоков в первой строке, определенных следующим образом: запишем все определенные выше матрицы $C_i$ в соответствующие блоки $M_{1i}$, а затем сложим с соответствующими матрицами $C'_i$.
	
	Таким образом, мы свели приближенное решение нелинейной проблемы собственных значений (\ref{problem}) к вычислению собственных значений матрицы $M$, что можно сделать применяя стандартный QR-алгоритм \cite{GVL}. Тогда приближенные решения проблемы (\ref{problem}) выражаются через собственные значения матрицы $M$ как $\lambda=\ln(\mu)/\delta$.
	
	Выберем для надежности из них определенное число ведущих (с максимальными вещественными частями) и уточним каждое из них методом последовательных линейных проблем \cite{CE,DNS_2017}, который заключается в следующем итерационном процессе: 
	\begin{gather} 
	\lambda_{j+1} = \lambda_j -\nu_j. \nonumber 
	\end{gather}
	
	Здесь $j$ --- номер итерации, а $\nu_j$ --- минимальное по абсолютной величине решение линейной проблемы собственных значений $A(\lambda_j)x = \nu B(\lambda_j)x$, где $B(\lambda)=dA/d\lambda$ и равна
	\begin{multline}
	B(\lambda) = I+\sum_j\omega_j L_{j}e^{-\lambda\omega_j}+\\+
	\begin{cases}
	\displaystyle
	(\frac{\omega_C e^{-(\lambda+\mu_C)\omega_C}}{\lambda+\mu_C}+\frac{1-e^{-(\lambda+\mu_C)\omega_C}}{(\lambda+\mu_C)^2})L_1, &|\lambda +\mu_C|\geq 1,\\
	\displaystyle
	\sum_{k=0}^{\infty} (-1)^k(\lambda+\mu_C)^{k-1}\omega_C^{k+1}(\frac{1}{k!}+\frac{1}{(k+1)!})L_1, &|\lambda +\mu_C|<1,
	\end{cases}
	\end{multline}
	где ${j\in \{U,C,E_2,E_1\}}$.
	
	На каждом Рис. \ref{1}---\ref{5} изображены $15$ приближенных и $7$ уточненных собственных значений с наибольшими вещественными частями в стационарных состояниях, значения переменных в которых приведены в Tабл.~\ref{Table2}. Синими маркерами <<o>> обозначены приближенные собственные значения  проблемы (\ref{problem}), полученные с помощью рациональной аппроксимации. Синими маркерами <<+>> обозначены собственные значения, уточненные методом последовательных линейных проблем. Видно, что первое  и четвертое состояния неустойчивы, остальные 3 стационарных состояния устойчивые.
